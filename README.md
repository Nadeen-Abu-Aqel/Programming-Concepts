## 🎯 الهدف من هذا الريبو

- تقديم ملخصات واضحة وسريعة  
- مساعدة الطلاب والمبتدئين  
- التحضير للمقابلات التقنية  

---

## 👥 مين ممكن يستفيد؟

- 👨‍🎓 طلاب تخصصات الحاسوب

- 💼 الأشخاص اللي يتحضروا لمقابلات  
- ⚡ أي شخص بدو مراجعة سريعة
