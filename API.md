# API (Application Programming Interface)

## 📌 التعريف
الـ API هو وسيلة تسمح لبرنامجين مختلفين بالتواصل مع بعضهم البعض.

---

## 💡 الفكرة الأساسية
بدل ما تبني كل شيء بنفسك، تستخدم API جاهز يوفر لك خدمات أو بيانات.

---

## 🔄 كيف يعمل؟
أنت ترسل "طلب" (Request)  
والـ API يرجع لك "رد" (Response)

---

## 🔥 مثال بسيط
تطبيق طقس:
- أنت تضغط "عرض الطقس"
- التطبيق يطلب البيانات من API
- API يرجع درجة الحرارة

---

## 🧠 أمثلة على APIs
- Google Maps API  
- Weather API  
- Payment API  

---

## 🎯 استخداماته
- جلب بيانات من الإنترنت  
- ربط أنظمة مع بعض  
- استخدام خدمات جاهزة  

---

## ⚠️ ملاحظات
- API مش برنامج كامل  
- هو مجرد وسيط بين الأنظمة  

---

## 📚 الخلاصة
API هو طريقة سهلة لربط البرامج مع بعض بدون تعقيد.
