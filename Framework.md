# Framework

## 📌 التعريف
الـ Framework (إطار العمل) هو مجموعة أدوات وقواعد جاهزة تساعد المبرمج في بناء التطبيقات بشكل أسرع وأسهل، بدون الحاجة لكتابة كل شيء من الصفر.



## 💡 الفكرة الأساسية
بدل ما تبني البرنامج من البداية:
                                                                                                                                     - Framework يعطيك هيكل (Structure) جاهز
- وانت تضيف عليه الكود الخاص فيك



## 🔄 كيف يعمل؟
الـ Framework يتحكم في تدفق البرنامج (Control Flow)  
وأنت تكتب الكود داخل هذا الإطار

👉 هذا يسمى:
                                                                                                                                           **Inversion of Control (IoC)**



## 🔧 ماذا يوفر؟
- كود جاهز (Reusable Code)
- تنظيم للمشروع
- أدوات لتسهيل التطوير
- تقليل الأخطاء
- تسريع العمل

---

## 🧠 أمثلة على Frameworks

### Web (Frontend)

- Vue.js

### Backend
- Spring Framework (Java)
- Django (Python)
- Express.js (Node.js)

### Mobile
- Flutter
- React Native

## ⚖️ الفرق بين Framework و Library

                                                                                                                                         || Framework | Library | Feature 
                                                                                                                                          --------|----------|--------|
| التحكم | هو يتحكم بالكود | أنت تتحكم |
| الاستخدام | هيكل كامل | جزء صغير |
| مثال | Angular | jQuery |



## 🎯 متى نستخدم Framework؟
- لما يكون المشروع كبير
- لما بدك تنظيم للكود
- لما بدك تشتغل بسرعة
- لما تشتغل ضمن فريق


## ⚠️ ملاحظات
                                                                                                                                                           Framework مش لغة برمجة
- لازم تتعلم اللغة أولاً
- أحيانًا يكون معقد للمبتدئين


## 🔥 مثال بسيط
بدون Framework:
- تكتب كل شيء بنفسك (واجهة + منطق + ربط)

مع Framework:
- تستخدم أدوات جاهزة
- تركز فقط على منطق البرنامج



## 📚 الخلاصة
                                                                                                                 Framework هو أداة قوية توفر وقت وجهد، وتساعد في بناء تطبيقات منظمة وقابلة للتطوير.
