# 🧼 Clean Code

## 📌 التعريف
كتابة كود واضح، مفهوم، وسهل القراءة والصيانة.

## 💡 الفكرة الأساسية
الكود مش بس يشتغل… لازم يكون مفهوم

---

## 🔧 ماذا يوفر؟
سهولة القراءة  
سهولة التعديل  
تقليل الأخطاء  

---

## 🎯 متى نستخدمه؟
دائمًا  

---

## ⚠️ ملاحظات
- استخدم أسماء واضحة  
- لا تطول functions  
- لا تكتب كود معقد  

---

## 🔥 مثال

❌ سيء:
int x;

✅ جيد:
int userAge;

---

## 📚 الخلاصة
اكتب الكود كأن غيرك رح يقرأه 👍
